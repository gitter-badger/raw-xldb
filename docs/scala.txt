INTRODUCTION TO SCALA

* Course on Functional Programming in Scala:
  https://www.coursera.org/course/progfun

Excellent course but covers only the functional programming part, e.g.
immutable data structures.

* Book on Scala:
  http://www.artima.com/shop/programming_in_scala_2ed

The Coursera course offers a discount coupon to order the book.

* Twitter Scala School:
  http://twitter.github.com/scala_school/

Twitter uses Scala and this is the course that new employees follow.


RELEVANT PUBLICATIONS

* "Lightweight Modular Staging: A Pragmatic Approach to Runtime Code Generation
  and Compiled DSLs"

Describes a library-based multistage programming approach that allows users
to define custom domain-specific languages and optimize them.

* "Language Virtualization for Heterogeneous Parallel Computing"

Two example applications of Scala's language virtualization for parallel
computing.
